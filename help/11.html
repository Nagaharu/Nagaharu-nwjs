文字シートについて


文字シートは、文章をデザインして配置するためのシートです。

設定項目の説明

行間 (「フォント」フォルダ )
行間を設定します。
文章が1行の場合は無視されます。

文字間 (「フォント」フォルダ )
文字間隔を設定します。

フォントサイズ (「フォント」フォルダ )
フォントの表示サイズを設定します。

フォント (「フォント」フォルダ )
表示に使用するフォントを指定します。
搭載されているフォントの詳細は、付録A.内蔵フォントについて をご覧下さい。

フォント色 (「フォント」フォルダ )
クリックすると、色を選択するためのカラーパレットが表示されます。

透明度 (「フォント」フォルダ )
1.0.1 Stableから追加された機能です。
シートの透明度を設定します。0(透明)～1(不透明)の範囲で設定します。

左揃え・中央揃え・右揃え (「文字装飾」フォルダ )
シート枠内の文章を指定の方向で揃えます。

太字・斜体・下線 (「文字装飾」フォルダ )
「太字と下線」といった複数の装飾処理も可能です。
Word等と違い、シート枠内の全ての文章が装飾されるので注意して下さい。

前面へ移動・背面へ移動・最前面・最背面 (「文字装飾」フォルダ )
画像と文字を重ねる時など、重なりの順序を変更する時にご利用下さい。

縦書き (「文字装飾」フォルダ )
シート枠内の文章を縦書きに変更します。
もう一度クリックすると、横書きに戻ります。

シートの削除 (「文字装飾」フォルダ )
選択中の文字シートを削除します。
