# Nagaharu-nwjs

###This software is Japanese-language only. Sorry:(

###概要
簡単DTPソフトウェア「Nagaharu」のnode-webkit版です。
###使用方法
最も簡単な方法を記述しておきます。

1、リポジトリをZIPファイルとしてダウンロード

2、「Nagaharu-nwjs-master」フォルダをNode-webkitにぶっ込む
