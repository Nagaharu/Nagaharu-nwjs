# Nagaharu-nwjs

## This software is Japanese-language only. Sorry:(

## 概要

簡単文書編集ソフトウェア「Nagaharu」のNW.js版です。

## 使用方法

最も簡単な方法を記述しておきます。

http://nwjs.io から最新のNW.jsを入手して下さい。

1、リポジトリをダウンロード

2、「Nagaharu-nwjs-master」フォルダをNW.jsで読み込む
